# stock-ticker-project

## virturalenv set-up
1. create a vitural enviroment
	> ```virtualenv tickerenv -p python3```
2. Install required packages
	> ```pip install -r requirements.txt```
3. Add virtualenv to .gitignore
	> ```vim .gitignore```
	> ```tickerenv```
